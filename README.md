# Banco_Ecommerce

Projeto de análise de dados utilizando SQL para simular um sistema de e-commerce.

## Objetivo

Realizar análises de vendas e clientes utilizando consultas SQL.

## Funcionalidades

- Cadastro de clientes
- Cadastro de produtos
- Registro de pedidos
- Relatórios de vendas
- Análise de ticket médio
- Produtos mais vendidos

## Tecnologias

- SQL
- MySQL
- PostgreSQL

## Estrutura do Projeto

```txt
database/
├── schema.sql
├── inserts.sql
└── queries.sql
```

## Consultas Realizadas

- Total gasto por cliente
- Produtos mais vendidos
- Ticket médio
- Quantidade de pedidos por cliente
